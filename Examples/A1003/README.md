# Açıklama #
<p>
Celsius'tan Kelvin'e Dönüşüm Formülü:<br>
K = C + 273.15<br>
Celsius'tan Fahrenheit'a Dönüşüm Formülü:<br>
F = (C * (9/5)) + 32<br>
</p>
